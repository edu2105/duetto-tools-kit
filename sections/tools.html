🧐Let's go through the current supported tools

Duetto API
HTNG Validator

The HTNG Validator is a tool that will compare any HTNG message type (such as Reservations, Inventory, Blocks, etc) agains an schema based on Duetto HTNG implementation.

Use it
Online Tool
XML To CSV

Instead of using public websites with production or sensitive data, better use a private website where you feel comfortable and secure. This tool will let you convert an XML payload with 1 nested level to a CSV file format that you can download.

Use it
Online Tool
Rate Sum

When checking an specific booking day, you may want to quickly calculate the rate sum total. This tool will allow you to upload a tsv file and will calculate the rate total automatically.

Use it
Online Tool
Mews Workaround

When receiving 408 Request Timout error from Mews, you can use this workaround to retrieve reservation messages.
